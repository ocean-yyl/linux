2，使用trap命令调试shell脚本
语法:trap 'command' signal
command---表示捕获信号后要执行的命令
signal---shell脚本执行中产生的3个伪信号。EXIT,ERR,DEBUG

可以使用DEBUG信号。在每条命令执行之前，都会触发信号。这实现：跟踪变量的值的变化。
例如：
#捕获信号变量
trap 'echo "before execute line"$LINENO,a=$a,b=$b,c=$c"' DEBUG

#待调试文本
a=1
if [ "$a" -eq 1]
then
        b=2
else
        b=1
fi
c=3

echo "end"
